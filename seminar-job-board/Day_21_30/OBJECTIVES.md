# Days 21–30 — Objectifs et Enseignements

## Objectifs

- Développer une plateforme de recrutement complète intégrant toutes les compétences acquises.  
- Concevoir et implémenter une API REST complète.  
- Créer une interface utilisateur moderne et responsive.  
- Gérer une base de données relationnelle complexe.

## Enseignements

- **Architecture full-stack** : Intégration frontend/backend, séparation des responsabilités
- **API REST** : Conception d'interfaces robustes, méthodes HTTP, format JSON
- **Base de données** : Modélisation relationnelle, requêtes SQL avancées, optimisation
- **Sécurité web** : Protection CSRF, validation des données, sanitisation, authentification
- **Gestion de projet** : Planification, livrables, tests, documentation
- **Développement professionnel** : Standards de code, collaboration, déploiement
