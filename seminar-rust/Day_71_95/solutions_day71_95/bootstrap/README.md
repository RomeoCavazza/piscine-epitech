# Rust Bootstrap — Days 71–76

This directory contains the initial exercises and the first asynchronous server built with Rust.

## Projects

- **[bootstrap_tasks](bootstrap_tasks/)**: Core logic exercises (Ownership, Borrowing, Lifetimes).
- **[bootstrap_server](bootstrap_server/)**: A multi-threaded, asynchronous server using `tokio` and `axum`.

## Key Concepts

- **Ownership & Borrowing**: Mastering memory safety without a garbage collector.
- **Pattern Matching**: Using `match` and `if let` for robust error handling.
- **Traits & Generics**: Implementing zero-cost abstractions.
- **Asynchronous Runtime**: Understanding the `tokio` event loop.
- **HTTP Handling**: First steps with `axum` and `serde` for JSON processing.
