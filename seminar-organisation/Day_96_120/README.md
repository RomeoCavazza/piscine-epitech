# Days 96–120 — Organizational Analysis & Strategic Design

- **Objectives**: [OBJECTIVES.md](OBJECTIVES.md)
- **Project Deliverables**: [solutions_day96_120/](solutions_day96_120/)

## Project Overview

**25 days** of organizational theory applied: auditing real-world companies (Conservatoire, Epitech, Agence Revolvr), identifying dysfunction patterns, designing target organizational models, and planning distributed expansion scenarios.

---

## Core Topics

### Sprint 1 (Days 96–110)
**Company Organizational Audits**

Three organizations analyzed in parallel:
- **Conservatoire André Navarra** (music education institution)
- **Epitech Paris** (tech school)
- **Agence Revolvr** (design/consulting agency)

Each audit delivers:
- Organizational charts (management, departments, teams)
- Internal process documentation
- Survey responses from stakeholders

### Sprint 2 (Days 111–115)
**Comparative Analysis**

Cross-company analysis identifying:
- Common organizational patterns
- Strengths vs. weaknesses
- Single Points of Failure (SPOF)
- Optimization opportunities

### Sprint 3 (Days 116–120)
**Strategic Design**

Deliverables:
- **Consolidated Target Model**: Ideal organizational blueprint combining best practices
- **Geographical Expansion Scenario**: Remote/distributed office model
- **Decision Process Definition**: Adapted governance for the target org

## Grading Rubric

| Criterion | Description |
|-----------|-------------|
| `data-relevance` | Collected data visualizes the organization's structure |
| `data-reliability` | Data sources are cited and reliable (from HR/management) |
| `data-involvement` | Students proactively gathered data |
| `chart-board` | Organizational chart of management/executive team |
| `chart-departments` | Chart showing departments and interactions |
| `chart-team` | Chart of own team and internal relationships |
| `processes` | Documentation of procedures (onboarding, meetings, workflows) |
| `analyse-op` | Analysis based on operational criteria (roles, hierarchy, teamwork) |
| `analyse-org` | Analysis based on organizational criteria (org type, centralization) |
| `analyse-dysf` | Identification of ≥2 dysfunctions (redundancies, bottlenecks) |
| `analyse-sol` | Proposal of ≥2 concrete solutions |
| `analyse-role` | Understanding own role and ability to influence |
| `compare` | Cross-org comparative analysis highlighting strengths/weaknesses |
| `speculate-think` | Document explaining growth reasons and perspectives |
| `speculate-draw` | Org model showing how the company could evolve |
| `speculate-manage` | Definition of decision-making process for target org |
| `presentation` | Professional presentation of project with support materials |
| `perfection` | Validation of all preceding criteria |

## Key Lessons

- **Organizational Patterns**: Functional vs. matrix vs. flat structures and their trade-offs
- **Information Flow**: How organizational design affects communication efficiency
- **Decision Dynamics**: Centralized vs. distributed decision-making
- **SWOT Analysis**: Identifying critical strengths and weaknesses
- **Process Mapping**: Visualizing workflows and identifying bottlenecks
- **Strategic Design**: Balancing structure with growth and operational capacity
- **Remote Models**: Challenges and solutions for distributed workforces
