# Reference Summaries

The `*.txt` files in this directory are short reference summaries used by
`bookworm.py --eval-summary` to compute ROUGE scores for the 21-book collection.

They are locally maintained, paraphrased summaries written for this project from
standard editorial knowledge of the works and checked against the corresponding
Wikipedia article openings. The reference type reported by the CLI is
`"wikipedia"` because the evaluation follows the project plan's Wikipedia-based
reference-summary convention.

Wikipedia text is available under CC BY-SA 4.0:
https://creativecommons.org/licenses/by-sa/4.0/
